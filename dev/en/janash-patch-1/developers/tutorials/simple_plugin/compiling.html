Attention

You may not be in the habit of using black and flake8, but we strongly recommend that you use the Makefile in SEAMM projects and use make lint install test religously as you develop. At first you may have a lot of format issues, but will quickly learn (be trained?) to write clean code in the style used in all SEAMM modules. In a large project such as SEAMM it is important that the code be similar throughout. We have set up the environment for black and flake8 to ensure consistency and a standard style for SEAMM. So please use the Makefile!
