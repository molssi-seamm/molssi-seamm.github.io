.. _dashboard:

*********
Dashboard
*********

The Dashboard is a key part of SEAMM that the users may interact with,
in addition to the :ref:`graphical user interface (GUI) <gui>`.

.. image:: /images/tutorial_2/initial_dashboard.png
	    :align: center
	    :alt: Dashboard

As a web-service, the Dashboard allows monitoring and managing
the executed :term:`jobs <job>` stored in the :term:`job datastore`.
It also enables the users to query the :term:`job manager` for
information regarding the running jobs.
